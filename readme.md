# Learn It, Love It Inc. - Aplicación de Temas Educativos

## Descripción
Esta es una aplicación web donde los usuarios pueden **agregar, actualizar, eliminar y votar por temas de aprendizaje**. La lista se reordena automáticamente según la cantidad de votos. Está construida usando **Node.js**, **Express**, **EJS** y **JavaScript puro**, cumpliendo con los requerimientos obligatorios del challenge.  

Este proyecto sigue la arquitectura **MVC** para mantener el código organizado y escalable.

---

## Tecnologías utilizadas
- Node.js
- Express.js
- EJS (Motor de plantillas)
- JavaScript puro (frontend)
- HTML / CSS
